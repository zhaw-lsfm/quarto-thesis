# ZHAW LSFM Thesis Template

A Quarto template implementing ZHAW LSFM thesis formatting guidelines using Typst.

## Installation

```bash
quarto use template zhaw-lsfm/quarto-thesis
```

## Documentation

Full documentation: <https://zhaw-lsfm.github.io/quarto-thesis-docs>

## Repository Structure

- `_extensions/zhaw-lsfm/` - Quarto extension
- `template.qmd` - Example thesis document
- Supporting files (bibliography, cover image)
